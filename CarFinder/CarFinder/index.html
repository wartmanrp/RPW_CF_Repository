Front End

AngularJS

The front end for this Car Finder application uses a combination of AngularJS and Ui-bootstrap

Built upon a basic foundation of HTML5, CSS3, and Twitter Boostrap, I also used AngularJS to define a controller and allocate 2 way data binding to elements within the page.

ui-bootstrap, an extension of AngularJS, was used to implement the image carousel, as well as the accordions for recall information.

If you are having trouble viewing recall information, try viewing the 1998 Ford Ranger, or 2009 Maserati Gran Turismo.

Back End

Web API + Stored Procedures

The back end for this Car Finder application uses ASP.NET's Web API framework as well as calls to a MS SQL database.

All car information is gathered from a database using stored procedures in MS SQL server. This serves to create a consumable RESTful API service which is then used by the application.You can view the API documentation by clicking here.

Carfinder also pulls from two other API's to provide information to the user. The first is the NHTSA recall database, to provide recall information. The second is Bing's image search API, which provides the images for the carousel.

All of this database and API information is then bound to Angular elements within the page to form a fully responsive, cross browser, single page application.
